# Timeline

本文件用于按月份记录个人成长过程。建议每月更新一次，内容包括学习内容、项目进展、技能提升、遇到的问题、解决方法和阶段性成果。

## 2026-07

### 学习内容

- C 语言基础复习：指针、结构体、函数模块化。
- STM32 外设学习：GPIO、UART、Timer、PWM。
- 机器人学基础：坐标变换、正运动学、自由度分析。
- 自动控制基础：系统响应、PID 控制思想。

### 项目进展

- 建立 Mechatronics-Learning-Portfolio 仓库结构。
- 开始整理手术机器人项目的机构设计、运动学和控制系统笔记。
- 建立激光雷达物流小车项目文档框架，准备补充硬件连接和调试记录。

### 技能提升

- 熟悉 GitHub 学习档案的组织方式。
- 初步建立工程项目复盘模板。
- 强化 Markdown 文档表达和技术关键词整理。

### 遇到的问题

- 项目资料分散，缺少统一记录入口。
- 对运动学、控制系统和嵌入式模块之间的关系理解还不够系统。
- 调试过程容易只记录结果，忽略现象、假设和验证步骤。

### 解决方法

- 使用统一模板记录项目、课程和调试问题。
- 将项目按机械、控制、嵌入式、算法分模块整理。
- 每周写一次学习日志，每月整理一次时间线。

### 阶段性成果

- 完成个人技术作品集初始结构。
- 明确未来学习路线和核心项目方向。
- 建立可持续维护的学习记录方法。
