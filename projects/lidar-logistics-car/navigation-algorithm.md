# Navigation Algorithm

## 目标

记录激光雷达物流小车的自动寻迹、路径跟踪和导航算法学习过程。

## 基础思路

1. 获取 LiDAR 或其他传感器数据。
2. 提取与路径、墙面、障碍物或目标点相关的信息。
3. 计算小车当前位置与目标路径的偏差。
4. 根据偏差调整左右轮速度。
5. 持续反馈修正，保持稳定运动。

## 可选算法

- 阈值判断。
- 比例控制。
- PID 路径跟踪。
- Pure Pursuit 基础思想。
- 障碍物检测与紧急停车。
