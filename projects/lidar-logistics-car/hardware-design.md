# Hardware Design

## 目标

记录智能仓储搬运机器人的硬件组成、结构设计、装配思路和后续需要补充的参数。硬件部分主要包括四麦克纳姆轮底盘、三轴机械臂、主控平台、下位机、传感器和电源 / 走线方案。

## 硬件系统组成

| 模块 | 选型 / 方案 | 作用 | 状态 |
| --- | --- | --- | --- |
| 主控平台 | Raspberry Pi 4B | ROS2 上位机、建图导航、任务调度 | 已使用 |
| 底盘下位机 | STM32F407 | 底盘电机控制、编码器读取、里程计输出 | 已使用 |
| 机械臂下位机 | STM32C8T6 | 机械臂坐标解算、行程限位、驱动输出 | 已使用 |
| LiDAR | RPLIDAR A1M8 | 二维激光建图和导航感知 | 已使用 |
| IMU | MPU6050 | 姿态数据输入，用于多传感器融合 | 已使用 |
| 编码器 | 轮式编码器 | 轮速 / 里程计反馈 | 已使用 |
| 移动底盘 | 四麦克纳姆轮 | 横移、原地旋转和全向移动 | 已装配 |
| 机械臂 | Y 轴伸缩 + Z 轴升降 + Yaw 转向 | 桌面目标抓取 | 已验证 |

## Chassis Design

底盘采用四麦克纳姆轮独立驱动方案，适合室内仓储场景中的横移、转向和狭窄通道通行。

### 设计重点

- 使用四个麦克纳姆轮实现全向移动能力。
- 使用铝合金底板兼顾轻量化、刚度和加工可行性。
- 预留电机、LiDAR、控制板、电池、机械臂和走线通道。
- 输出切割零件工程图，标注尺寸、孔位、板厚和表面处理要求。
- 在 CAD 装配中检查麦轮、螺母、电机支架和底板之间的干涉。
- 通过有限元分析检查底板强度，确认最大应力低于材料屈服强度。

### 装配流程

1. 底盘钻孔。
2. 固定电机。
3. 安装麦克纳姆轮。
4. 安装支撑柱和上层结构。
5. 固定控制板、电池和传感器。
6. 整理走线并检查运动空间。

### 已发现并修正的问题

| 问题 | 影响 | 处理 |
| --- | --- | --- |
| 麦轮与螺母存在局部干涉 | 可能影响轮子正常转动 | 调整连接件和安装位置 |
| 控制板和传感器孔位需要预留 | 后期装配可能返工 | 在底板设计阶段统一规划孔位 |
| 走线空间有限 | 影响调试和维护 | 预留走线通道并分层安装 |

## Arm Hardware

机械臂安装在移动底盘上方，用于执行目标抓取。结构自由度包括：

- `Yaw`: 机械臂整体转向。
- `Z`: 竖直方向升降。
- `Y`: 水平方向伸缩。

已记录的控制范围：

| 轴 | 范围 / 说明 |
| --- | --- |
| Z 轴 | 0-200 mm 行程 |
| 半径 / 伸缩 | 0-180 mm |
| 控制方式 | STEP / DIR + PWM |

机械臂控制细节见：[arm-control.md](./arm-control.md)

## Sensor Layout

| 传感器 | 安装目的 | 数据用途 |
| --- | --- | --- |
| RPLIDAR A1M8 | 安装于车体上方，扫描室内平面环境 | `/scan`，用于建图和导航 |
| MPU6050 IMU | 安装于底盘或主控附近 | `/imu/data_raw`，用于姿态估计 |
| 轮式编码器 | 与底盘电机 / 轮组配合 | `/wheel/odom`，用于里程计 |
| 相机 / 点云传感器 | 面向目标物体 | 目标识别与空间定位 |
