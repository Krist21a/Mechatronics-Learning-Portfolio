# Hardware Design

## 目标

记录激光雷达物流小车的硬件组成、连接方式、供电方案和结构设计。

| 模块 | 作用 | 状态 |
| --- | --- | --- |
| 主控板 | 控制逻辑与数据处理 | 待补充 |
| LiDAR | 距离测量和环境感知 | 待补充 |
| 电机驱动 | 驱动左右轮电机 | 待补充 |
| 编码器 | 速度 / 里程反馈 | 待补充 |
| 电源模块 | 给主控、传感器和电机供电 | 待补充 |
| 底盘结构 | 安装各硬件模块 | 待补充 |
