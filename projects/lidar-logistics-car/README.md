# LiDAR Logistics Car 激光雷达物流小车

## Project Overview

本项目用于记录基于激光雷达和嵌入式控制的自动寻迹 / 导航物流小车学习过程。项目重点包括硬件系统搭建、传感器数据读取、电机控制、路径跟踪算法和调试复盘。

当前项目处于学习与整理阶段，文档会持续补充实际连接方式、代码结构、调试数据和问题解决过程。

## Background

自动物流小车是机器人和自动化系统中的典型应用，涉及移动底盘、传感器感知、路径规划、运动控制和嵌入式系统。激光雷达可用于环境感知、距离测量、障碍物检测和路径辅助判断。

## Hardware System

计划记录的硬件模块包括：

- 主控板：STM32 或其他嵌入式控制器。
- 传感器：LiDAR、编码器、IMU、限位或避障传感器。
- 执行机构：直流电机 / 减速电机 / 舵机。
- 驱动模块：电机驱动板、电源管理模块。
- 车体结构：底盘、轮组、安装支架和传感器固定结构。

详细记录见：[hardware-design.md](hardware-design.md)

## Navigation / Path Following Algorithm

导航与寻迹算法当前以基础路径跟踪为主，后续逐步扩展：

- 基于距离 / 角度信息的方向修正。

- 路径丢失后的恢复逻辑。
