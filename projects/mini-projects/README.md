# Mini Engineering Projects 小型工程项目

本目录用于记录课程设计、实验、小型机构、控制系统和编程练习项目。

## 记录范围

- C / Python 编程练习。
- STM32 外设实验。
- 传感器读取与数据处理。
- 电机控制小实验。
- 机械结构或 CAD 小设计。
- OpenCV / AI 小实验。

## 项目记录格式

建议每个小项目至少包含：

- 项目背景。
- 实现目标。
- 硬件 / 软件组成。
- 核心代码或算法说明。
- 调试问题。
- 实验结果。
- 复盘与改进。
