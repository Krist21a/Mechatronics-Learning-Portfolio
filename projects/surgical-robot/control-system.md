# Control System

## 目标

记录手术机器人控制系统学习过程，包括控制架构、电机控制、运动指令、安全限制和反馈闭环。

## 控制层级

```text
目标动作
  -> 轨迹 / 速度规划
  -> 关节控制量
  -> 电机驱动信号
  -> 编码器 / 传感器反馈
  -> 误差修正
```

## 当前关注点

- PID 控制在速度环和位置环中的应用。
- 采样周期对控制稳定性的影响。
- 限位保护、急停逻辑和异常状态处理。
- 控制算法与机械结构约束之间的关系。

## 待实验内容

- 单电机速度闭环控制。
- 编码器位置读取与角度换算。
- 简单位置控制实验。
- 运动命令与安全边界限制。
