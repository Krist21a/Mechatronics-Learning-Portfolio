# Kinematics Analysis

## 目标

记录手术机器人机构的运动学学习与分析过程，建立从关节变量到末端位姿的基础理解。

## 分析内容

- 坐标系建立。
- 齐次变换矩阵。
- Forward Kinematics 正运动学。
- Inverse Kinematics 逆运动学。
- Jacobian 雅可比矩阵。
- 工作空间与奇异位形。

## 当前进展

- 正在学习旋转矩阵和齐次变换矩阵。
- 准备以简化机械臂模型为例推导正运动学。
- 后续将把机构参数与具体结构方案对应起来。
