# Microcomputer Interface

本目录用于记录微机原理与接口技术课程笔记。

## 笔记目录

- [第 2 章 Cortex-M3 处理器知识笔记总结](./cortex-m3-chapter-02.md)
- [第 3 章 STM32F1 系列微控制器知识笔记总结](./stm32f1-chapter-03.md)

## 学习重点

- 微处理器基本结构。
- Cortex-M3 体系结构、寄存器、工作模式和堆栈。
- STM32F1 系列型号命名、系统架构和内部资源。
- AHB / APB 总线、RCC 时钟系统和外设时钟使能。
- BOOT 启动配置、最小系统和 SWD / JTAG 调试接口。
- 存储器与 I/O 接口。
- 存储器映射、位带操作和外设访问。
- 异常、中断系统和 NVIC 优先级。
- 串口通信。
- 定时器 / 计数器。

## 项目关联

- STM32 外设配置。
- Cortex-M3 异常响应、SysTick 和中断优先级配置。
- STM32F1 最小系统、时钟树和启动模式排查。
- 串口调试和传感器通信。
- 嵌入式控制系统设计。
