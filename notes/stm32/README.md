# STM32 Notes

本目录用于整理 STM32 学习笔记和实验记录。

## 主题

- 工程创建与时钟配置。
- Cortex-M3 内核基础、异常中断和 NVIC。
- STM32F1 型号命名、系统架构、启动配置和最小系统。
- Thumb-2 汇编基础、启动文件和寄存器级调试。
- GPIO、EXTI、Timer、PWM。
- UART、I2C、SPI。
- DMA 和中断。
- 编码器与电机控制。
