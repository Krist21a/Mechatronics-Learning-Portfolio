# Roadmap

本路线面向机电一体化 / 机器人方向本科生，目标是把课程知识、工程工具和项目实践连接起来，形成可持续积累的学习路径。

## 第一阶段：编程基础与工程工具

### 学习目标

- 掌握 C 语言、Python 基础语法和常见工程写法。
- 熟悉 Git、Markdown、VS Code / IDE、串口调试助手等工具。
- 建立代码记录、笔记整理和项目复盘习惯。

### 关键知识点

- C 语言：变量、数组、指针、结构体、函数、文件组织。
- Python：数据处理、脚本自动化、绘图分析。
- Git：commit、branch、remote、README 编写。
- 工程规范：目录结构、命名、注释、版本记录。

### 对应项目应用

- 建立本学习仓库。
- 编写小型 C 语言练习和 Python 数据分析脚本。
- 为课程实验建立标准记录模板。

## 第二阶段：嵌入式系统与传感器

### 学习目标

- 掌握 STM32 基础外设和常见通信接口。
- 能够读取传感器数据，并进行初步滤波和异常处理。
- 建立嵌入式项目的硬件连接、软件驱动和调试记录方法。

### 关键知识点

- GPIO、EXTI、Timer、PWM、ADC。
- UART、I2C、SPI。
- 编码器、IMU、激光雷达、限位开关、光电传感器。
- 中断、DMA、状态机和通信协议。

### 对应项目应用

- 激光雷达物流小车传感器读取与电机控制。
- 手术机器人控制原型中的按键、限位、编码器和通信模块。
- 传感器接口专题笔记整理。

## 第三阶段：电机控制与自动控制

### 学习目标

- 理解开环控制与闭环控制的区别。
- 掌握 PID 控制的基本原理、参数调试和实际问题处理。
- 能够完成电机速度 / 位置控制的基础实验。

### 关键知识点

- 传递函数、系统响应、稳定性分析。
- PID 控制、采样周期、误差、积分饱和。
- 编码器测速、PWM 调速、电机驱动保护。
- 直流电机、步进电机、舵机、BLDC 基础。

### 对应项目应用

- 物流小车左右轮速度闭环控制。
- 手术机器人关节运动控制实验。
- 电机控制调试日志与参数记录。

## 第四阶段：机器人学与运动学

### 学习目标

- 掌握机器人位姿描述、坐标变换和运动学建模。
- 能够对机械臂或手术器械腕部结构进行自由度分析。
- 理解轨迹规划与运动控制之间的关系。

### 关键知识点

- 齐次变换矩阵、旋转矩阵、欧拉角。
- Forward Kinematics、Inverse Kinematics。
- Jacobian、奇异性、速度映射。
- 轨迹规划、插补、速度限制。

### 对应项目应用

- 手术机器人 EndoWrist 类机构运动学分析。
- 小型机械臂或关节结构建模。
- 用 Python / MATLAB 进行运动学仿真。

## 第五阶段：计算机视觉与人工智能

### 学习目标

- 掌握 OpenCV 基础图像处理流程。
- 理解相机标定、目标检测和视觉测量的基本方法。
- 建立 AI 与机器人感知任务之间的联系。

### 关键知识点

- 图像读取、滤波、边缘检测、形态学处理。
- Camera Calibration、畸变校正、坐标映射。
- Object Detection、特征提取、深度学习基础。
- 数据集、训练、评估和部署概念。

### 对应项目应用

- 视觉识别小实验。
- 物流小车视觉辅助导航扩展。
- 手术场景下器械识别或图像辅助定位的概念学习。

## 第六阶段：综合项目实践

### 学习目标

- 将机械结构、嵌入式、电机控制、传感器和算法整合到完整系统。
- 能够用工程化方式表达项目：需求、架构、实现、调试、复盘。
- 形成可展示的 GitHub 项目作品集。

### 关键知识点

- 系统架构设计。
- 硬件选型与接口规划。
- 软件模块划分与状态机。
- 调试日志、版本管理、实验数据记录。
- 项目总结、风险分析和后续改进。

### 对应项目应用

- Surgical Robot Project 手术机器人项目。
- LiDAR Logistics Car 激光雷达物流小车。
- Mini Engineering Projects 小型工程项目合集。
