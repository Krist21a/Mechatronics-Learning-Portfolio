# GitHub Topics 建议

建议在 GitHub 仓库页面添加以下 topics，方便别人通过技术方向理解这个作品集仓库。

## Recommended Topics

- `mechatronics`
- `robotics`
- `embedded-systems`
- `stm32`
- `motion-control`
- `control-engineering`
- `computer-vision`
- `opencv`
- `surgical-robotics`
- `lidar`
- `python`
- `c-language`

## 使用建议

- 优先选择与仓库长期方向高度相关的 topics。
- topics 数量不需要太多，保持清晰即可。
- 如果后续加入 ROS、AI 模型或更多机械设计内容，可以继续补充 `ros`, `machine-learning`, `mechanical-design` 等标签。
